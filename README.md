# Convertify

Convertify is an all-in-one web platform that provides useful digital tools for students, creators, developers, and everyday users in one modern and responsive application.

🌐 Live Demo: https://convertivy.lovable.app/

---

## 🚀 Features

### Productivity Tools
- Study Timer (Pomodoro)
- JSON Formatter
- Unit Converter

### File & Utility Tools
- File to Link
- QR Code Generator
- Image Compression

---

## 💳 Credits System

- Every new user starts with **20 free credits**
- Premium tools consume **2 credits per use**
- Free tools:
  - Pomodoro Timer
  - JSON Formatter

Users can purchase additional credits through the payment system.

---

## 📱 Web + Mobile Experience

Convertify is designed to work on:
- Desktop
- Tablets
- Mobile devices

The platform also supports:
- PWA installation
- Mobile-friendly UI
- Capacitor integration for Android/iOS apps
